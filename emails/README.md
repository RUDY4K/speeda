# Email Sequences — Speeda

5 إيميلات بالعربي للتجربة المجانية لمدة 14 يوم. تستخدمها مع أي ESP (Resend, Postmark, Mailchimp, SendGrid).

## التوقيت

| اليوم | الإيميل | الهدف |
|---|---|---|
| Day 0 | `01-welcome.md` | ترحيب فوري + تفعيل الحساب |
| Day 1 | `02-day1-quick-win.md` | يربط Foodics ويشغّل أول رد |
| Day 3 | `03-day3-success-story.md` | يبني الثقة بقصة عميل ناجح |
| Day 7 | `04-day7-midpoint.md` | يستعرض النتائج الفعلية للمستخدم |
| Day 12 | `05-day12-decision.md` | اختيار الباقة + خصم |
| Day 14 | `06-day14-final.md` | آخر فرصة قبل انتهاء التجربة |

## النبرة

- ودود وغير مزعج
- مباشر، بدون فلسفة
- 100% عربي سعودي
- كل إيميل يطلب فعل واحد (CTA واحد بس)
- شخصي — يستخدم {{first_name}} و {{restaurant_name}}

## كيف تستخدمها

استبدل المتغيرات:
- `{{first_name}}` → اسم العميل
- `{{restaurant_name}}` → اسم المطعم
- `{{dashboard_url}}` → رابط لوحة التحكم
- `{{messages_count}}` → عدد الرسائل اللي ردّ عليها البوت
- `{{orders_count}}` → عدد الطلبات اللي دخلت Foodics

## ESP المقترح للمشروع

**Resend** (resend.com): مجاني للـ 3000 إيميل/شهر، سهل الربط مع Next.js، يدعم العربي بدون مشاكل.
